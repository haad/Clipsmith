## Decisions Made

- BezelViewModel reads UserDefaults.standard directly in navigateUp/Down — consistent with existing stickyBezel/pasteMovesToTop pattern; @AppStorage requires a SwiftUI view context
- removeDuplicates guard wraps the full dedup block unchanged — existing behaviour preserved when enabled, simply skipped when disabled
- bezelAlpha applied as `1.0 - bezelAlpha` to opacity — 0.25 default becomes 0.75 opacity (semi-transparent); higher slider values give more transparency
- savePreference=0 uses mainContext.delete(model:) synchronously on applicationWillTerminate — background ModelActor is unavailable after process termination begins
