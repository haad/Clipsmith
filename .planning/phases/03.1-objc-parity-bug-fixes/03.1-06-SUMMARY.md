## Decisions Made
- `kAXTrustedCheckOptionPrompt` accessed via string literal `"AXTrustedCheckOptionPrompt" as CFString` — the global `Unmanaged<CFString>` reference triggers Swift 6 shared-mutable-state error; string literal is the correct workaround
- s/S key handling placed before the `chars.lowercased()` switch — case-sensitive dispatch is required because both `s` and `S` would collapse to `"s"` after lowercasing
- Merge All uses `modelContext` directly in MenuBarView (same pattern as existing Clear All) — avoids needing ClipboardStore injection into the view layer
- Password-length filter reads UserDefaults directly in ClipboardMonitor — consistent with all other UserDefaults reads in the service layer
