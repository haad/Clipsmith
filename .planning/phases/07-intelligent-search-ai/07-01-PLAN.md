
    Algorithm (character-subsequence with consecutive bonus scoring, per RESEARCH.md Pattern 1):
    1. Guard empty query returns 1.0
    2. Lowercase both candidate and query
    3. Iterate through candidate characters; when current char matches current query char, advance query index and add consecutive bonus to score (bonus increments by 1.0 for each consecutive match)
    4. When chars don't match, decay consecutive bonus by 0.5 (min 0)
    5. If all query chars consumed, normalize score by dividing by ideal score (n*(n+1)/2 where n = query length)
    6. If NOT all query chars consumed, return nil (not a subsequence)

    ```swift
    func recomputeFilteredClippings() {
        guard !searchText.isEmpty else {
            filteredClippings = clippings

        let q = searchText

    }
    ```
