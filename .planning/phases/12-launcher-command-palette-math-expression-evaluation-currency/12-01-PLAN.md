    Create `Clipsmith/Services/ExpressionEvaluator.swift` as a single-file pure Swift service.

    Imports: `import Foundation` (only).

    Declare `nonisolated struct ExpressionEvaluator`.

    Private static stored properties:
    - `safeMathRegex: NSRegularExpression` compiled with `try!` from raw-string pattern `#"^[\d\s()\.\+\-\*/,eE]+$"#`. (The `*` is already in the class. `**` is `* *` adjacent — both are in the class. Do NOT add `^` to the safe set; preprocessing strips it first.)
    - `funcNames: [String]` = `["sqrt", "abs", "ceiling", "floor", "ln", "log", "exp"]`

    Public static methods:

    `@MainActor static func evaluate(_ rawText: String) -> Double?`
    Algorithm:
    1. Trim whitespace. If empty, return `nil`.
    2. Preprocess `^` → `**`: `replacingOccurrences(of: #"(?<![*])\^(?![*])"#, with: "**", options: .regularExpression)` — this avoids accidentally double-replacing if user already typed `**`.
    3. Build a `stripped` copy of the lowercased expression and remove each entry from `funcNames` (`.replacingOccurrences(of: fn, with: "")`). The remaining text must then match `safeMathRegex`. If it does not match, return `nil` WITHOUT touching `NSExpression`. This is the T-12-01 mitigation gate.
    4. Construct `let nsExpr = NSExpression(format: expr)` and call `nsExpr.expressionValue(with: nil, context: nil)`. Cast to `NSNumber` — return `nil` on cast failure.
    5. Extract `doubleValue`. Reject `isNaN` and `isInfinite` (covers division-by-zero per Pitfall 8). Return the `Double` otherwise.

    `nonisolated static func formatResult(_ value: Double) -> String`
    Algorithm (per D-06 and Pattern 3):
    1. If `value == value.rounded(.toNearestOrEven) && !value.isInfinite && abs(value) < 1e15`, treat as integer:

If the executor prefers `Locale.current`, document it in the SUMMARY but `en_US` keeps tests deterministic.), `maximumFractionDigits = 0`. Return `formatter.string(from: NSNumber(value: Int64(value)))` falling back to `"\(Int64(value))"`.
    2. Otherwise return `String(format: "%.6g", value)`.

    `nonisolated static func copyableResult(_ value: Double) -> String`
    Return `String(format: "%.10g", value)`. No thousands grouping — this is what gets copied to the clipboard so it remains paste-safe in code editors and spreadsheets.
