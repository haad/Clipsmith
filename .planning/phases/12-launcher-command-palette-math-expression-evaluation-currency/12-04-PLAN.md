       ```
       /// Copies the current command palette result to the clipboard without

       ///
       /// Flow per D-13:
       /// 1. No-op if commandResult is nil (Invalid expression — bezel stays open)
       /// 2. Write copyableValue to NSPasteboard.general
       /// 3. Set clipboardMonitor.blockedChangeCount = pasteboard.changeCount
       ///    so the monitor skips the cycle in which our write was recorded.
       /// 4. Set viewModel.showCopiedToast = true (CommandPaletteView animates the
       ///    "Copied ✓" overlay).
       /// 5. After 1.2 seconds, clear the toast and hide the bezel.
       ///
       /// Note: this does NOT call PasteService.simulateCmdV — D-13 specifies "copy
       /// to clipboard", not paste. Reusing PasteService here is an anti-pattern.
       func copyResult() {
           guard let result = viewModel.commandResult else { return }
           let pasteboard = NSPasteboard.general
           pasteboard.clearContents()
           pasteboard.setString(result.copyableValue, forType: .string)
           // T-12-02 mitigation: tell ClipboardMonitor to skip this changeCount
           clipboardMonitor?.blockedChangeCount = pasteboard.changeCount
           viewModel.showCopiedToast = true
           Task { @MainActor [weak self] in
               guard let self else { return }
               try? await Task.sleep(for: .seconds(1.2))
               self.viewModel.showCopiedToast = false
               self.hide()
           }
           logger.info("Command palette result copied to clipboard")
       }
       ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ClipboardMonitor → AppLaunchController.copyResult | The monitor must skip the pasteboard write that copyResult performs to avoid recording the result as a new clipping |
| Settings UI → CurrencyService.refreshRates | User-initiated network request to open.er-api.com; bounded to user click |
| Settings → NotificationCenter → AppDelegate.handleCurrencyRatesRefreshed | In-process notification (not cross-process); the payload is the bare name (no userInfo), so no untrusted data crosses the boundary |
| AppDelegate startup → CurrencyService.loadRates | File system read of bundled JSON + optional downloaded file |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-02 | Info Disclosure | AppLaunchController.copyResult → NSPasteboard | mitigate | `clipboardMonitor?.blockedChangeCount = pasteboard.changeCount` set IMMEDIATELY after `pasteboard.setString`. AppDelegate Task 2 injects the live `clipboardMonitor` instance (the `let clipboardMonitor = ClipboardMonitor()` at AppDelegate line 28). The human-verify checkpoint in Task 3 exercises the end-to-end flow and confirms the result does not appear in clipboard history. |
| T-12-03 | Tampering | CurrencyService.refreshRates triggered from Settings | mitigate | Plan 02 already gates the JSON decode through typed `Codable` before writing to disk; this plan only invokes the existing method. The new `.clipsmithCurrencyRatesRefreshed` notification carries no payload (object: nil, no userInfo) — AppDelegate's handler just calls `loadRates()` which re-runs the same Codable decode. No new attack surface. |
| T-12-04 | Tampering | commandPalettePrefix TextField | mitigate | Settings UI in Task 2 enforces: empty → reset to "="; >1 char → keep first; alphanumeric → reset to "=". The prefix is only used in `String.hasPrefix()` downstream (Plan 03), never interpolated into shell, eval, or SQL. |
