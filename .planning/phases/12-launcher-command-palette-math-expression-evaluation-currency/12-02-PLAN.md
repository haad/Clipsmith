    - Inject: `init(session: URLSession = .shared)`

    Methods:

    `func loadRates()` — synchronous (called from AppDelegate on launch; later plans wire it):
    1. Try `Data(contentsOf: downloadedRatesURL)` first. On success, decode `ExchangeRateResponse`, assign `rates = response.rates`, set `lastUpdated` from file modification date via `FileManager.default.attributesOfItem(atPath:)[.modificationDate] as? Date`.

    3. If both fail, log a warning and leave `rates` empty.

    `func refreshRates() async`:
    1. Set `isRefreshing = true`, `lastError = nil`. Use `defer { isRefreshing = false }`.
    2. Build URL: `URL(string: "https://open.er-api.com/v6/latest/USD")` — guard nil → set lastError and return.
    3. Wrap in do/catch:
       - `let (data, _) = try await session.data(from: url)`
       - Decode `ExchangeRateResponse` BEFORE writing to disk (so a malformed response never corrupts the cached file — T-12-03 defense in depth)
       - `try FileManager.default.createDirectory(at: downloadedRatesURL.deletingLastPathComponent(), withIntermediateDirectories: true)`
       - `try data.write(to: downloadedRatesURL, options: .atomic)`
       - Assign `rates = response.rates`, `lastUpdated = Date()`
    4. On any error: `lastError = error.localizedDescription`. Do NOT clear `rates`. Do NOT write to disk.

    `func convert(amount: Double, from: String, to: String) -> Double?`:
    1. `guard !rates.isEmpty else { return nil }`
    2. `let fromUpper = from.uppercased(); let toUpper = to.uppercased()`
    3. `guard let fromRate = rates[fromUpper], let toRate = rates[toUpper], fromRate != 0 else { return nil }`
    4. Return `amount / fromRate * toRate` (rates are USD-base; convert via USD)

    `static func isCurrencyQuery(from: String, to: String) -> Bool`:
    Returns true iff `from.count == 3 && to.count == 3 && from.uppercased().allSatisfy(\.isLetter) && to.uppercased().allSatisfy(\.isLetter)`.

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-03 | Tampering | `CurrencyService.refreshRates` JSON parsing | mitigate | `JSONDecoder().decode(ExchangeRateResponse.self, from: data)` with strongly-typed `Codable` struct (rates is `[String: Double]`, no arbitrary code). Decode happens BEFORE the file is written to disk — a malformed/malicious response never overwrites the cached file. Acceptance criteria verify this ordering. |
| T-12-03b | DoS | Currency API returns huge response | accept | URLSession ephemeral session and JSONDecoder both have implicit memory caps; the documented `rates` payload is < 10 KB. No additional mitigation. |
| T-12-04 | Tampering | UnitConversionService query regex | mitigate | The query regex restricts captured tokens to `[\w°]+` and the value to `[\d.,]+(?:[eE][+-]?\d+)?`. Anything else returns nil — no eval, no shell, no SQL. |
| T-12-SC | Tampering | npm/pip/cargo installs | accept | Phase 12 adds ZERO new SPM packages (verified in RESEARCH.md Package Legitimacy Audit). No package install tasks; no audit required. |
