    1. Create `FlycutSwift/Services/PromptLibraryStore.swift`:
       - Define `PromptInfo: Sendable, Identifiable` struct (mirrors SnippetInfo):
         ```swift
         struct PromptInfo: Sendable, Identifiable {
             let id: PersistentIdentifier
             let promptID: String        // stable slug from JSON
             let title: String
             let content: String
             let category: String
             let version: Int
             let isUserCustomized: Bool
             let isUserCreated: Bool
         }
         ```
       - Define `PromptDTO: Decodable, Sendable` struct for JSON parsing:
         ```swift
         struct PromptDTO: Decodable, Sendable {
             let id: String
             let title: String
             let category: String
             let version: Int
             let content: String
         }
         ```
       - Define `PromptCatalog: Decodable, Sendable` for the top-level JSON:
         ```swift
         struct PromptCatalog: Decodable, Sendable {
             let version: Int
             let prompts: [PromptDTO]
         }
         ```
       - Create `@ModelActor actor PromptLibraryStore` with methods:
         - `insert(id:title:content:category:version:isUserCreated:) throws`
         - `fetchAll() throws -> [PersistentIdentifier]` (sorted by title ascending)
         - `fetchByCategory(_ category: String) throws -> [PersistentIdentifier]`
         - `search(query: String) throws -> [PersistentIdentifier]`
           Search supports `#category` prefix syntax: if query starts with "#", extract category name (up to first space), filter by category, then search remaining text in title/content. If no "#" prefix, search title and content with localizedStandardContains.
         - `upsert(remote: PromptDTO) throws` — version-aware:
           Fetch by id. If existing: skip if isUserCustomized==true; skip if remote.version <= existing.version; otherwise update title/content/version/updatedAt. If not found: insert new.
         - `update(id: PersistentIdentifier, title: String, content: String) throws` — sets isUserCustomized=true, updates title/content/updatedAt
         - `revertToOriginal(id: PersistentIdentifier) throws` — sets isUserCustomized=false (next sync restores upstream content)
         - `delete(id: PersistentIdentifier) throws`
         - `prompt(for id: PersistentIdentifier) -> PromptInfo?`
         - `content(for id: PersistentIdentifier) -> String?`
       - Use Logger with category "PromptLibraryStore"
       - All mutations call `modelContext.save()` immediately after (PITFALL 3)

    2. Create `FlycutSwift/Services/TemplateSubstitutor.swift`:
       - `struct TemplateSubstitutor` (pure value type, no actor needed)
       - `static func substitute(in content: String, variables: [String: String]) -> String`
         Uses Swift Regex `/\{\{(?<variable>[^}]+)\}\}/` with `.replacing(_:with:)` closure.
         Trims whitespace from captured variable name before dictionary lookup.
         Returns original `{{...}}` if no match in variables dict (unknown vars left as-is).
       - `static func extractVariables(from content: String) -> [String]`
         Returns unique variable names found in the content (for UI display).
       - No clipboard reading here — that happens at paste time in the caller.
