2. In `applicationDidFinishLaunching`, after the existing service wiring (after step 4 where appTracker.start() and clipboardMonitor.start() are called), create the BezelController. The BezelController needs references to `pasteService` and `appTracker` for the paste-and-hide flow. Pass them during init or set them as properties after creation. The BezelController also needs the sharedModelContainer for @Query to work inside BezelView.

3. Replace the `activateBezel` hotkey handler:
```swift
KeyboardShortcuts.onKeyDown(for: .activateBezel) { [weak self] in
    Task { @MainActor in
        guard let self else { return }
        if self.bezelController.isVisible {
            self.bezelController.hide()
        } else {
            self.bezelController.show()
        }
    }
}
```

4. Replace the `activateSearch` hotkey handler:
```swift
KeyboardShortcuts.onKeyDown(for: .activateSearch) { [weak self] in
    Task { @MainActor in
        guard let self else { return }
        self.bezelController.showWithSearch()
    }
}
```

5. In `applicationWillTerminate`, add `bezelController.hide()` before other cleanup to ensure the global monitor is removed.

Do NOT change the existing service wiring logic (clipboardMonitor, clipboardStore, pasteService, appTracker). Only replace the two hotkey handler closures and add the bezelController property + initialization.
