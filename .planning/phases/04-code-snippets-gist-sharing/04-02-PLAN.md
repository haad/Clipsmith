    ```swift
    @MainActor @Observable
    final class GistService {
        private let tokenStore: TokenStore
        private let session: URLSession
        private let modelContext: ModelContext

        init(modelContext: ModelContext,
             tokenStore: TokenStore = TokenStore(),
             session: URLSession = .shared) { ... }

        func createGist(filename: String, content: String,
                        description: String, isPublic: Bool) async throws -> GistResponse { ... }

        func deleteGist(id: PersistentIdentifier) async throws { ... }

        static func languageExtension(for language: String?) -> String { ... }

        static let languageExtensions: [String: String] = [
            "swift": "swift", "python": "py", "javascript": "js",
            "typescript": "ts", "ruby": "rb", "go": "go",
            "rust": "rs", "kotlin": "kt", "java": "java",
            "bash": "sh", "c": "c", "cpp": "cpp",
            "sql": "sql", "html": "html", "css": "css",
            "json": "json", "yaml": "yaml", "xml": "xml",
            "markdown": "md"
        ]
    }
    ```

    Mark as `@Observable` so it can be injected into SwiftUI environment (same pattern as PasteService).

    `createGist` flow:
    1. Load token from TokenStore — throw `GistError.noToken` if nil
    2. Construct `CreateGistRequest` Encodable struct
    3. Build URLRequest with headers (Accept, Authorization Bearer, X-GitHub-Api-Version 2022-11-28, Content-Type)
    4. Call `session.data(for: request)` — async/await, Swift 6 clean
    5. Check HTTP 201 — throw `GistError.httpError(statusCode)` otherwise
    6. Decode `GistResponse` (id + html_url with CodingKeys)
    7. Copy URL to clipboard: `NSPasteboard.general.clearContents(); NSPasteboard.general.setString(response.htmlURL, forType: .string)` (GIST-04)
    8. Persist `FlycutSchemaV1.GistRecord(gistID:gistURL:filename:)` to modelContext
    9. Return GistResponse

    `deleteGist` flow:
    1. Fetch GistRecord from modelContext by PersistentIdentifier
    2. Call DELETE `https://api.github.com/gists/{gistID}` with Bearer token
    3. On HTTP 204 success, delete local GistRecord from modelContext

    Define `GistError: Error, LocalizedError` enum with cases: `noToken`, `httpError(Int)`, `networkError(Error)`.

    Use `OSLog.Logger` with category "GistService".

    **REFACTOR:** Extract URL construction into a private helper.
