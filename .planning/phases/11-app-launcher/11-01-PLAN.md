<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| filesystem→AppScannerService | Reading from /Applications, /System/Applications etc. — system-owned, trusted by macOS |
| UserDefaults→AppScannerService | Reading recentBundleIDs from UserDefaults — user-writable but only consumed for display ordering, never executed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01 | Tampering | AppScannerService.scanApps | mitigate | Only enumerate URLs from the five hardcoded whitelisted search paths (D-01). The search query (when added in Plan 02) is never passed to FileManager or NSWorkspace. |
| T-11-03 | Information disclosure | AppScannerService.recordLaunch | accept | Bundle IDs in UserDefaults are public macOS identifiers (e.g. com.apple.Safari) — not credentials. Local-only storage. |
</threat_model>
