<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| KeyboardShortcuts hotkey → AppDelegate handler | System-trusted IPC; only fires when user-registered hotkey is pressed |
| Settings UserDefaults appLauncherEnabled → hotkey guard | User-writable flag; only effect is gating the bezel show |
| menu bar Button → .clipsmithOpenAppLauncher → openAppLauncherFromMenu | Internal NotificationCenter; no external sender can reach this without process injection |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01 | Tampering | AppDelegate.openAppLauncherFromMenu | accept | NotificationCenter is process-local; external tampering would require code execution in our process. |
| T-11-02 | Elevation of privilege | KeyboardShortcuts.onKeyDown(.appLauncher) handler | mitigate | Handler calls only `appLaunchController.show()`. Show() does not launch any app — it only opens the bezel. Actual app launch goes through `launchSelected()` which uses URLs from `AppScannerService` scan results only. |
| T-11-03 | Information disclosure | recordLaunch UserDefaults | accept | Bundle IDs are public macOS identifiers; not sensitive. |
</threat_model>
