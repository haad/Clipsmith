<task type="auto">
  <name>Task 1: Export/Import UI in Settings and MenuBar</name>
  <files>FlycutSwift/Views/Settings/GeneralSettingsTab.swift, FlycutSwift/Views/MenuBarView.swift, FlycutSwift/App/AppDelegate.swift</files>
  <action>
    **Settings > General (GeneralSettingsTab.swift):**

    Add two new Notification.Name extensions (in MenuBarView.swift alongside existing ones):
    - `.flycutExportHistory` and `.flycutImportHistory`

    In the "Data" section of GeneralSettingsTab, after the "Remove duplicate clippings" toggle, add:

    ```swift
    HStack {
        Button("Export History...") {
            NotificationCenter.default.post(name: .flycutExportHistory, object: nil)
        }
        Button("Import History...") {
            NotificationCenter.default.post(name: .flycutImportHistory, object: nil)
        }
    }
    ```

    Notification bridge pattern is used (same as .flycutShareAsGist) because GeneralSettingsTab is a SwiftUI view that doesn't have access to AppDelegate's clipboardStore. AppDelegate handles the actual export/import.

    **MenuBarView.swift:**

    Add Export/Import menu items after "Clear All" and before the Divider:
    ```swift
    Button("Export History...") {
        NotificationCenter.default.post(name: .flycutExportHistory, object: nil)
    }
    Button("Import History...") {
        NotificationCenter.default.post(name: .flycutImportHistory, object: nil)
    }
    ```

    **AppDelegate.swift:**

    Add two NotificationCenter observers in applicationDidFinishLaunching (after existing observers):
    ```swift
    NotificationCenter.default.addObserver(self, selector: #selector(handleExportHistory), name: .flycutExportHistory, object: nil)
    NotificationCenter.default.addObserver(self, selector: #selector(handleImportHistory), name: .flycutImportHistory, object: nil)
    ```

    Add `@objc private func handleExportHistory()`:
    1. Task { @MainActor in ... } to ensure main thread for NSSavePanel
    2. Call ClipboardExportService.exportHistory(from: clipboardStore) to get Data
    3. Create NSSavePanel with allowedContentTypes: [.json], nameFieldStringValue: "flycut-history.json", canCreateDirectories: true
    4. Call panel.begin { response in ... } -- if .OK, write data to panel.url
    5. On error, show NSAlert with error message

    Add `@objc private func handleImportHistory()`:
    1. Task { @MainActor in ... }
    2. Create NSOpenPanel with allowedContentTypes: [.json], allowsMultipleSelection: false
    3. Call panel.begin { response in ... } -- if .OK:
    4. Read Data from panel.urls.first
    5. Show NSAlert asking "How would you like to import?" with buttons: "Merge with existing" (default), "Replace all history", "Cancel"
    6. If merge or replace: call ClipboardExportService.importHistory(into: clipboardStore, from: data, merge: isMerge)
    7. Show success alert: "Imported N clippings"
    8. On error, show NSAlert with error message

    Import UniformTypeIdentifiers in AppDelegate for .json content type.
