<objective>
Build the three runtime services: ClipboardMonitor (pasteboard polling with password/transient filter), PasteService (CGEventPost Cmd-V injection with timing), and AppTracker (previous-app tracking via workspace notifications).

Purpose: These are the invisible services that make the clipboard manager actually work. The monitor watches for new clipboard content, the paste service injects it back, and the app tracker ensures paste goes to the right app. Without these, the stored clippings have no way in or out.

Output: Three service classes ready for wiring, plus unit tests for the testable parts (monitor filtering, pasteboard write verification).
</objective>

    - `var clipboardMonitor: ClipboardMonitor?` — set during wiring so PasteService can update `blockedChangeCount` after pasteboard write (prevents self-capture per Pitfall 1).
    - `func paste(content: String, into previousApp: NSRunningApplication?) async`:
      1. Guard `AXIsProcessTrusted()` — if false, log error and return early. Do NOT call `AXIsProcessTrustedWithOptions(prompt: true)` — that steals focus (Pitfall 3).
      2. `NSPasteboard.general.clearContents()`
      3. `NSPasteboard.general.setString(content, forType: .string)` — plain text ONLY, no RTF (CLIP-06).
      4. Set `clipboardMonitor?.blockedChangeCount = NSPasteboard.general.changeCount` — block self-capture.
      5. If `previousApp` is non-nil and not terminated: `previousApp.activate(options: .activateIgnoringOtherApps)` — use `NSRunningApplication.activate(options:)`, NOT the deprecated `NSApp.activate(ignoringOtherApps:)`.
      6. `try? await Task.sleep(for: .milliseconds(200))` — 200ms minimum (Pitfall 4). Do NOT reduce below 150ms.
      7. Call `injectCmdV()`.
    - `private func injectCmdV()`:
      1. `CGEventSource(stateID: .combinedSessionState)` — guard non-nil.
      2. Create keyDown + keyUp for virtualKey 9 (V key).
      3. Set keyDown.flags to `[.maskCommand, CGEventFlags(rawValue: 0x000008)]` (secondary command bit per original Flycut).
      4. `CGEventPost(.hid, keyDown)` then `CGEventPost(.hid, keyUp)`.
      5. Log "Cmd-V injected" at debug level.
    - Use `os.Logger` with category "PasteService".
