Features

One tool for everything you type, copy, and reuse.

📋

Clipboard History

Everything you copy, always one shortcut away. Adaptive polling keeps it responsive without draining your battery.

  • Quick Actions Press Tab to transform: uppercase, lowercase, title case, trim, URL encode/decode, JSON pretty print, Markdown code block
  • Fuzzy search subsequence matching with consecutive-bonus scoring finds what you need even with typos
  • Keyboard navigation arrows, j/k, 0-9 numeric jump, Page Up/Down, Home/End. Paste with Return, never touch the mouse
  • Password filtering auto-skips likely credentials from 1Password, LastPass, and configurable length heuristics
  • Smart deduplication duplicates move to top with fresh timestamps instead of cluttering your history
  • Source app tracking see which app each clipping came from. Star favorites, export/import full history as JSON
  • GitHub Gist sharing press G to share any clipping as a public or private Gist instantly
  • Plain text paste strips all rich formatting (RTF, HTML) for predictable paste behavior. No surprise fonts
  • Non-activating panel the bezel never steals focus from your current app. Copy, pick, paste — your cursor stays where it was
📝

Code Snippets

A persistent library of reusable code blocks with syntax highlighting. Not another clipping — a named, tagged, searchable snippet you keep forever.

  • Syntax highlighting 20+ languages: Swift, Python, JavaScript, TypeScript, Go, Rust, Ruby, Java, Kotlin, C, C++, Bash, SQL, JSON, YAML, HTML, CSS, Markdown, and more
  • Multi-tag system assign multiple tags per snippet for flexible cross-cutting organization. Filter by tag or language
  • Full-text search search across name, content, and tags simultaneously with locale-aware matching
  • Dedicated editor monospaced font, auto-save on edit. Undo/redo, no auto-correct interference, no smart quotes
  • Keyboard access press ⇧⌘S to open the snippet editor from anywhere. Search, select, paste — same flow
  • Separate from history snippets persist independently and never get trimmed. Your clipboard history can rotate while snippets stay permanent
💬

Prompt Library

40+ curated AI prompts for coding, analysis, research, and DevOps — with template variables that auto-fill from your clipboard.

  • Template variables use {{clipboard}}, {{selected_text}}, and custom variables. Auto-substituted on paste
  • Category filtering Tab to cycle categories, or type #coding review to search within a category. Use #my for your own prompts
  • 3-second flow hotkey → type a few characters → Enter. The prompt is pasted with your clipboard content already injected
  • Remote sync sync prompts from a configurable URL. Share a team prompt library via JSON hosted anywhere
  • Create your own add custom prompts alongside the built-in library. User prompts are flagged separately and never overwritten by sync
  • Browse online all built-in prompts are also available in the web-based Prompt Library to browse and copy
📖

Documentation Browser

Search 789+ DevDocs documentation sets from a single hotkey. Download docs for offline use — never leave your editor to look something up.

  • 789+ documentation sets Python, Go, Rust, JavaScript, React, Node.js, Ruby, CSS, HTML, and hundreds more from devdocs.io
  • Prefix search type python:map or go:fmt to scope search to a single doc. Partial names work too — “py” matches “python~3.12”
  • Offline support selectively download docs. Search and viewing work entirely offline after download
  • Dark & light themes documentation renders in WKWebView with custom CSS matching your system appearance
  • Split-pane UI search results on the left, rendered HTML on the right. Resizable, with position remembered across sessions
  • Fuzzy matching same scoring engine as clipboard search. Top 100 results ranked by relevance, then by name length

And more
