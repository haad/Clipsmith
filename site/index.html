Clipsmith

A clean and simple clipboard manager for developers

Keyboard-first Open source macOS native

Download from GitHub

Features

Everything you need from a clipboard manager, nothing you don't.

📋

Clipboard History

Automatically saves everything you copy. Scroll back through your clipboard history and paste any previous item.

⌨

Keyboard-First Navigation

Navigate your clipboard with arrow keys, paste with Return. Designed for developers who keep their hands on the keyboard.

🔎

Fuzzy Search

Instantly find any clipping by typing. Fuzzy matching surfaces what you need even with partial or misspelled queries.

📝

Code Snippets

Save frequently used code as named snippets. Organize into folders and paste with a keyboard shortcut.

☁

GitHub Gist Sharing

Share any clipping as a GitHub Gist with one keystroke. Perfect for quickly sharing code with teammates.

💬

Prompt Library

Built-in collection of curated AI prompts organized by category. Browse, search, and paste prompts for ChatGPT, Claude, and more.

⚡

Quick Actions

Transform clipboard text instantly: uppercase, lowercase, trim whitespace, sort lines, encode URLs, and more.

🔒

Privacy First

All data stays on your Mac. No cloud sync, no analytics, no tracking. Your clipboard is yours.

🛠

Open Source

Fully open source under the MIT license. Inspect the code, contribute features, or fork it for your own needs.

Keyboard Shortcuts

Every action is a keystroke away.

Navigation

  • Open Clipsmith ⇧⌘V
  • Previous clipping ↑
  • Next clipping ↓
  • Paste selected ⏎
  • Search clippings /
  • Dismiss Esc

Actions

  • Delete clipping ⌦
  • Star / unstar S
  • Share as Gist G
  • Quick actions Tab
  • Preview Space
  • Shortcut help ?

Organization

  • All clippings ⌘1
  • Starred ⌘2
  • Snippets ⌘3
  • Prompts ⌘4
  • Snippet folders ⌘N
